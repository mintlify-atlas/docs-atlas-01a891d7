## Overview

miPOS is a Python-based point of sale (POS) system featuring a graphical user interface built with Tkinter. It provides inventory management, sales tracking, reporting, and AI-powered analytics capabilities.

<Note>
**Project Name**: miPOS (mipunto -)  
**Main File**: inicio.py  
**Location**: `~/workspace/source/proyectos/mipunto -/`  
**GUI Framework**: Tkinter
</Note>

## System Architecture

### Main Application Window

```python
import tkinter as tk
from inventario import *
from ai import *
from reporte import *
from registro import *

ventana = tk.Tk()
ventana.geometry("1000x500")  # Width x Height
```

## Installation

### Dependencies

```bash
# Tkinter is included with Python
# No additional packages required for basic functionality
python --version  # Ensure Python 3.6+
```

<Note>
Tkinter comes pre-installed with most Python distributions. On Linux, you may need:
```bash
sudo apt-get install python3-tk  # Ubuntu/Debian
sudo yum install python3-tkinter  # CentOS/RHEL
```
</Note>

### Setup Steps

<Steps>
  <Step title="Clone/Copy Project">
    Copy the `mipunto -` directory to your workspace:
    ```bash
    cp -r mipunto\ - /path/to/workspace/
    ```
  </Step>
  
  <Step title="Verify Files">
    Ensure all module files are present:
    ```bash
    ls mipunto\ -/
    # Should show: inicio.py, inventario.py, ventas.py, reporte.py, ai.py, registro.py
    ```
  </Step>
  
  <Step title="Run Application">
    ```bash
    cd mipunto\ -
    python inicio.py
    ```
  </Step>
</Steps>

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Window doesn't appear | Check Tkinter installation: `python -m tkinter` |
| Buttons not responding | Verify function imports in `inicio.py` |
| Module not found | Ensure all `.py` files are in same directory |
| Window too small/large | Adjust `ventana.geometry("WIDTHxHEIGHT")` |
